# Lady Fresa — Reporte Abril 2026

Análisis operativo y financiero del mes de Abril 2026 para Lady Fresa.

## 🌐 Ver el reporte

Una vez activado GitHub Pages, el reporte se publica en:
`https://[tu-usuario].github.io/ladyfresa-abril-2026/`

## 📊 Contenido del reporte

- **Resumen ejecutivo** con KPIs del mes
- **Ventas diarias** y comportamiento por día de la semana
- **Distribución por hora** del día con mapa de calor
- **Mix de productos** y ranking por unidades
- **Análisis de tamaños, bases y toppings**
- **Combinaciones más frecuentes** (qué compran juntos)
- **Canal de venta** (POS vs Uber/Didi/Rappi)
- **Comparativo Marzo vs Abril**
- **Hallazgos clave y recomendaciones para Mayo**
- **Estado de resultados** y P&L
- **Margen por producto**
- **Detalle de los 102 movimientos** de gastos

## 🚀 Cómo publicarlo en GitHub Pages

1. Crear un nuevo repositorio en GitHub (puede llamarse `ladyfresa-abril-2026` o lo que prefieras)
2. Subir todos los archivos de esta carpeta al repo
3. Ir a **Settings → Pages**
4. En **Source** elegir la rama `main` (o `master`) y la carpeta `/ (root)`
5. Guardar — en 1-2 minutos el reporte está publicado
